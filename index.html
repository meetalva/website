Design and engineering, tied together

You can start with a minimal set of components to sketch concepts like wireframes and do fast iterations with your development team to create and enrich components, from atoms to modules and entire pages with production-ready code.

Cross­functional single source of truth

Maintaining a separate design library next to the developer’s component library is quite weird. That’s why Alva lets you design with exactly the same code components your engineers use for the final application.

Based on React components

All components are based on React, that means you are designing with real code instead of just static shadow copies in a former design tool. But if youre new component is not already developed, you can just drop in a PNG out of Sketch (Constraint driven design)

Real DOM

Instead of classical layers you can see the live DOM order here.

Responsive

In order to see the responsive behaviour so you can always adjust the viewport.

Interactive

The web is interactive, so we should design interactively, too. Alva allows designing complex interactive experiences.
